# Windsurf — Conductor Activation

Windsurf reads project rules from `.windsurfrules` at the repo root.

## Setup

```bash
cp conductor-core/activation/windsurf/.windsurfrules .windsurfrules
```

Open Windsurf. Start a Cascade chat. Say "Activate Conductor."
