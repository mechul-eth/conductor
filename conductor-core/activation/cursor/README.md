# Cursor — Conductor Activation

Cursor reads project rules from `.cursorrules` at the repo root.

## Setup

```bash
cp conductor-core/activation/cursor/.cursorrules .cursorrules
```

Open Cursor. Start a new chat. Say "Activate Conductor."

Cursor follows `conductor-core/activation/FIRST_RUN.md` on first activation and the behavioral contract (`.cursorrules`) on every subsequent session.
